# CANOPY 網站結構說明

目前正式網站架構與開發範圍請以 [`docs/website-structure.md`](docs/website-structure.md) 為唯一規格來源。

本階段核心決策：

1. 植物商品固定為十六種。
2. `data/plants-detailed-v2.json` 是唯一植物商品資料來源。
3. 商品頁包含商品規格、適生環境、完整照護、介質比例、配送與安全說明。
4. 移除抽象分級，以介質類型取代原本的商品篩選與卡片標籤。
5. Header 導覽 Hover 使用底部青草綠橫線。
6. 移除個人植物管理與照護紀錄功能，登入後體驗延後重新規劃。
